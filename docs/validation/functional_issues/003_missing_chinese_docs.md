# Issue #3: 缺少中文文档

## 严重程度
**中** - 影响中文用户体验

## 问题描述
SDK 目前只有英文文档，缺少中文版本的：
- README.md
- API 参考文档
- 使用示例
- 故障排除指南

## 影响
- 中文开发者使用门槛高
- 需要频繁切换翻译工具
- 理解错误消息困难
- 社区推广受限

## 缺失的文档
1. **中文 README**：完整的中文介绍和快速开始
2. **API 参考中文版**：所有公共接口的中文说明
3. **示例代码中文注释**：examples/ 目录下的代码注释
4. **错误消息中文版**：提供中文错误消息选项

## 建议实现
1. 创建 `README_zh.md` 中文版 README
2. 在 `docs/zh/` 目录下创建中文文档
3. 添加 `lang` 参数控制错误消息语言
4. 提供中英双语示例代码

## 功能优先级
**中** - 对于中文用户社区很重要
